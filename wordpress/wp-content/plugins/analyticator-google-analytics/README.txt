Google Analytics by Analyticator is a Simple and easy to use Google analytics tracking plugin for WordPress.

== Description ==

Google Analytics by Analyticator plugin have very easy configuration for setting up Google analytics for a WordPress site. It doesn't ask for Authentication from Google analytics to fetch the data.

Just add your GA ID manually and setup tracking. Advanced settings are available too e.g Cross-domain tracking, Force SSL traffic, User-ID, Demographic & Interest Tracking etc

Simple and easy to use Google analytics tracking plugin for WordPress.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `google-analytics-analyticator` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Settings->Setup Google Analytics and enter your GA ID. 
1. Choose advanced settings options if you know what to do with them.
